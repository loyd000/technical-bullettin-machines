// 01 — System Overview
What Is This System?
Page 3

The AMTEC Agricultural Machinery Data system is an interactive web catalogue that lets users browse, search, and compare the technical specifications of agricultural machines tested by AMTEC. Data is pulled live from a published Google Sheets document, so no database configuration is required — updating the sheet is enough to update the website.

🗂️
Category Browser
Machines are grouped into categories (e.g. Four-Wheel Tractor, Hammer Mill). Each category has its own sheet tab.
🔍
Search & Filter
Full-text search across all spec fields. Filter by brand. Results update live as you type.
⚖️
Side-by-Side Compare
Select any two machines from the same category to compare all specs in a table. Highlight only the rows that differ.
🖨️
Print Bulletin
One-click printable technical bulletin with AMTEC/UPLB header, machine photo, and full spec table.
🔗
Shareable Links
Deep-link directly to a category or a specific machine comparison. The URL updates as you navigate.
📥
CSV Export
Download any comparison as a spreadsheet-ready CSV file for offline analysis or reporting.
Data Flow
📊
Google Sheets
Admin edits the sheet. Data is published as a public CSV feed.
⚙️
WordPress Plugin
Plugin fetches CSV, parses it in the browser, and renders the UI.
👁️
Visitor / End User
Browses, searches, compares, and prints — no login required.
💡
Tip: Data is cached for 5 minutes in the browser. If you have just updated the Google Sheet, click the ↺ Refresh button on the category page to pull the latest data immediately.

// 02 — Home Page
Machine Categories
Page 4

When you open the page, you see the Home View — a grid of category cards, one for each type of agricultural machinery. Each card shows a photo (if set in the sheet) and the category name.

amtec.uplb.edu.ph/machinery-data/

What you see

  • Category photo — pulled from the photo_url column of the Categories sheet. If empty, a placeholder icon is shown.
  • Category label — the machine type name (e.g. "Four-Wheel Tractor").
  • Arrow → — animates on hover to indicate the card is clickable.

How to navigate

1
Click a category card
The URL updates to #category=four-wheel-tractor and the category view loads.
2
Go back
Click ← All Categories (top-left) to return to the category grid at any time.
ℹ
Deep links work! You can bookmark or share a URL with #category=four-wheel-tractor appended to take someone directly to a specific category without needing to click through the home page.

// 03 — Browsing Specifications
The Machine List
Page 5

After selecting a category, you land on the Category View. The Machine List tab is active by default, showing a card for every machine in the sheet.

amtec.uplb.edu.ph/machinery-data/#category=four-wheel-tractor

Anatomy of a Machine Card

  • Photo — loaded from the photo_url spec row. Shows a placeholder icon if missing.
  • Brand & Model — the bold heading at the top of the card.
  • Type — the machine's category type (e.g. "Two-Wheel Drive").
  • Test Requested By — shown in amber below the type.
  • Show detailed specifications — expands the full spec table (see next section).
  • Print — opens a printable one-page bulletin (see §05).
💡
The item count at the top right (e.g. 12 items) updates live when you apply search or brand filters. Use it to confirm your filter is working.

// 03.2 & 03.3 — Search, Filter & Expand
Finding & Reading Specs
Page 6

Searching Machines

Search box
1
Type in the search box
Searches name, brand, model, category, and every spec value. Results narrow as you type (200ms debounce).
2
Expand a card while searching
Matched text is highlighted in yellow inside the spec table.

Filtering by Brand

Brand filter

Use the Filter by brand dropdown to show only machines from one manufacturer. Combine it with the keyword search to narrow results further. Select All brands to remove the filter.

Expanding Detailed Specifications

Spec table (expanded)
1
Click "Show detailed specifications"
The full spec table slides open below the card. Specs are grouped by section (e.g. Engine, Transmission, Dimensions).
2
Read section headers
Amber section banners (e.g. ENGINE SPECIFICATIONS) group related rows.
3
Click again to collapse
The button label toggles to "Hide detailed specifications".

// 04 — Compare Machines
Side-by-Side Comparison
Page 7

The Compare Machines tab lets you place two machines from the same category next to each other and compare every specification row by row.

Compare Machines tab

Setting Up a Comparison

1
Click the "Compare Machines" tab
The first two machines in the category are pre-selected automatically. Machine photos appear side-by-side at the top.
2
Choose Machine A and Machine B
Use the two dropdowns to select any two different machines. The comparison table updates instantly.
3
Scroll through the spec table
All specification rows appear in the same order as the source sheet. Section headers divide related groups of specs.

// 04.3 — Share & Export
Sharing & Exporting Comparisons
Page 8

Copy Shareable Link

🔗 Copy Link button
1
Click "Copy Link"
The current page URL (which includes the two machine IDs) is copied to your clipboard. The button briefly shows ✓ Copied!
2
Paste and share
Anyone who opens the link will see the same two machines pre-selected in the Compare tab.
ℹ
The URL format looks like:
…#category=four-wheel-tractor&compare=kubota-l3408,ford-6610

Export to CSV

📥 Export CSV button
1
Select two machines
Both Machine A and Machine B dropdowns must have a value selected.
2
Click "Export CSV"
A .csv file downloads automatically, named compare-[id-a]-vs-[id-b].csv.
3
Open in Excel or Google Sheets
The file has three columns: Specification, Machine A name, Machine B name — one row per spec.

// 05 — Print Bulletin
Printing a Machine Bulletin
Page 9

Each machine card has a Print button that generates a formatted one-page specification bulletin complete with AMTEC and UPLB logos, machine photo, and the full spec table — ready to print or save as PDF.

Print preview window

How to print

1
Find the machine card
Scroll to the machine or search for it in the Machine List.
2
Click "🖨 Print"
A new browser tab opens and the system print dialog appears automatically.
3
Print or Save as PDF
In the print dialog, choose your printer or select "Save as PDF" to generate a digital copy.

Bulletin layout

  • Header row — UPLB logo left, AMTEC logo right, institution name center.
  • Machine identity — Brand (amber), Model (large), Category below.
  • Photo — shown on the left if a photo_url is set in the sheet.
  • Spec table — all specifications with section banners.
⚠
Pop-ups must be allowed on this site. If the print window does not open, look for a blocked pop-up notification in the browser address bar and click Always allow.

// 06 — Refreshing Data
The ↺ Refresh Button
Page 10

The system caches machine data in the browser for 5 minutes to reduce load on Google Sheets. If you have just edited the spreadsheet and want to see the changes immediately, use the Refresh button.

Refresh button — top of category page

Steps

1
Navigate into a category
The Refresh button only appears on the category view, not the home page.
2
Click "↺ Refresh"
The button is disabled and shows "↺ Refreshing…" while the sheet is being fetched.
3
Machine list reloads
The list updates with the latest data from Google Sheets. The 5-minute cache resets.

When to use Refresh

  • You just added a new machine to the sheet and want it to appear.
  • You updated a specification value and need to verify the change.
  • You added or changed a photo URL and want to see the new image.
  • You renamed a category in the Categories sheet.
ℹ
Google Sheets has its own publication delay of a few minutes after you save. Even after clicking Refresh, allow 2–3 minutes for the Google Sheets CSV to update if changes are very recent.

// 07 — Admin Guide
Managing the Google Sheet
Page 11

All content is managed entirely inside Google Sheets. There are two types of sheet tabs: a Categories index sheet and one machine data sheet per category.

Categories Sheet

Tab name must be exactly Categories. Each row is one category.

Column Description
id URL slug (e.g. four-wheel-tractor)
label Display name (e.g. Four-Wheel Tractor)
gid Sheet tab GID (found in the sheet URL)
icon Emoji icon (optional)
photo_url Full URL to category card image

Machine Data Sheet

Each machine is a column. Each specification is a row. Column A is the spec label; columns B, C, D … are the machines.

💡
Section headers: Leave the machine columns empty for a row to make it a section banner (e.g. "Engine Specifications").
💡
Photos: Add a row with label Photo URL and paste the full image URL in each machine's column.
⚠
The sheet must be published to the web (File → Share → Publish to web) as CSV for the system to read it.

Adding a New Machine

1️⃣
Open the sheet tab
Go to the correct category tab in Google Sheets.
2️⃣
Add a new column
Insert a new column to the right of existing machines.
3️⃣
Fill in spec values
Enter Brand, Model, and all spec values in the corresponding rows.
4️⃣
Save & Refresh
Wait 2–3 min, then click ↺ Refresh on the website.

// 08 — Troubleshooting
Common Issues & Fixes
Page 12
Problem Likely Cause Fix
Category grid shows "No machine categories" Categories sheet not published or tab name wrong Verify the tab is named exactly Categories and the sheet is published as CSV.
"No machine data found for this category" Wrong GID in the Categories sheet, or machine tab is empty Copy the GID from the sheet URL and paste it into the gid column. Confirm the tab has data rows.
Machine photos not showing URL is not a direct image link, or image is not publicly accessible Use a full https:// URL that ends in an image extension. Upload to WordPress Media Library for reliable URLs.
Print button does not open a window Browser is blocking pop-ups Click the pop-up blocked icon in the address bar → "Always allow pop-ups from this site".
Sheet edits not appearing after Refresh Google Sheets publication has a delay Wait 2–3 minutes after saving, then click Refresh again.
Compare table shows "N/A" for one machine Machines have different numbers of spec rows Ensure all machine columns in the sheet have the same number of rows as the first column.
Spec section headers not appearing Section rows have data in machine columns A row becomes a section header only if ALL machine columns in that row are empty. Clear any accidental values.
